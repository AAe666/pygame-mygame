# 🎮 Pygame 游戏集合

各种 Pygame 小游戏的集合,按游戏类型分类管理。

## 📁 分类说明

- **01-shooting**: 射击类游戏(外星人入侵、太空射击等)
- **02-platform**: 平台跳跃类(马里奥、冒险岛等)
- **03-puzzle**: 益智类(俄罗斯方块、扫雷等)
- **04-strategy**: 策略类(塔防、棋类等)
- **05-rpg**: 角色扮演类
- **06-sports**: 运动类(贪吃蛇、乒乓球等)
- **common**: 各游戏共享的代码模块

## 🚀 快速开始

cd 01-shooting/alien-invasion
.\venv312\Scripts\activate
python main.py

## 📚 开发规范

- 每个游戏使用独立的虚拟环境
- 统一的项目结构
- 每个游戏都要有 README.md
- 使用 Git 进行版本控制
